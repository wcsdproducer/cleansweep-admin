# Preview Project Workflow

Follow these steps to run and test the local development server for CleanSweep Admin.

## Step 1: Start Dev Server
Run the local next development server:
```bash
npm run dev
```
*(By default, this server listens on port `9002`)*

## Step 2: Open and Verify Web UI
Open your browser to:
[http://localhost:9002/](http://localhost:9002/)

## Step 3: Access Admin Dashboard
Navigate to the admin portal/dashboard to verify user authentication views:
[http://localhost:9002/dashboard](http://localhost:9002/dashboard)
