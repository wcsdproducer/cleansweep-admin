# Build Project Workflow

Follow these steps to install dependencies and build the CleanSweep Admin Next.js project.

## Step 1: Install Dependencies
Run `npm install` to install all project dependencies:
```bash
npm install
```

## Step 2: Run Typecheck
Verify compile-time TypeScript type safety:
```bash
npm run typecheck
```

## Step 3: Compile and Build
Generate the optimized production build:
```bash
npm run build
```
