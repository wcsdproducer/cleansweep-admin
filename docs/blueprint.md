# **App Name**: StudioVault Admin

## Core Features:

- Dashboard Overview: Provide a high-level summary of workspace activity, key metrics, and system status at a glance.
- Project Management: Enable administrators to view, create, update, and delete projects within the 'studio-3673070449-f277c' database.
- User Management: Allow administrators to manage user accounts, including roles, permissions, and status within the system.
- Data Browsing & Editing: Provide an interface to browse and directly edit raw data within the 'studio-3673070449-f277c' database tables.
- AI-Powered Report Summary Tool: Utilize generative AI to provide a concise, natural language summary of selected project activities or user behavior data points from the database.
- Database Connection Monitoring: Display the connection status and basic performance metrics for the 'studio-3673070449-f277c' database.

## Style Guidelines:

- Primary color: A sophisticated deep violet (#662999) conveying professionalism and digital elegance. Chosen for an 'admin' tool managing valuable 'studio' data, it provides a sense of calm authority for effective data management.
- Background color: A very light, desaturated violet (#ECE6F0) that ensures content readability and maintains visual continuity with the primary hue.
- Accent color: A vibrant blue (#3D52F5) providing a clear, energizing contrast for calls to action, important notifications, and highlighting active states. This analogous hue adds dynamism while maintaining harmony with the primary palette.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and highly legible characteristics, ensuring optimal readability across all data displays and navigation elements in an admin interface.
- Minimalist, line-based icons for all interactive elements and navigation to ensure clarity and avoid visual clutter on data-heavy screens.
- Clean, modular dashboard layout with clear sectioning and responsive design to support efficient data overview and management across different screen sizes. Utilize cards and tables for structured data display.
- Subtle, fast transitions for state changes, navigation, and data loading indicators to enhance user experience without causing distraction in a functional administrative tool.
